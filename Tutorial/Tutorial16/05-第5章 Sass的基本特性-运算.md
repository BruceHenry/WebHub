### 5-4 [Sass运算]除法
- 如果数值直接除需要外面加小括号
- 变量除不需要加小括号
- 如果还有别的运算符也不需要加

```
.box {
  width: (100px / 2);  
}

.box {
  width: 100px / 2 + 2in;  
}

$width: 1000px;
.item {
  width: $width / 10;  
}
```
